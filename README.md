# Network Traffic Analysis with Wireshark

## Objective
Capture and analyze network traffic to identify normal vs suspicious activity.

## Tools
- Wireshark (Windows)

## Steps
1. Installed Wireshark
2. Captured network traffic on active interface
3. Applied filters: http, dns
4. Identified suspicious or unusual traffic
5. Saved capture and screenshots for documentation

## Observations
- Observed DNS queries to multiple domains
- Spotted unusual repeated connections to external IPs

## Skills Gained
- Basic packet capture
- Filtering network traffic
- Recognizing anomalies
